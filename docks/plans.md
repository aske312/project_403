## 13. Definition of Ready (DoR)
Задача готова к работе, если:
* есть описание
* понятен результат
* известны затрагиваемые сервисы/API
* определены права доступа
* нет блокеров по инфраструктуре
* есть критерии проверки

---
## 14. Definition of Done (DoD)
Задача завершена, если:
* работает функционально
* интегрирована с системой
* обновлены env/config при необходимости
* добавлены или обновлены проверки по риску задачи
* пройдены lint/build/backend smoke-checks
* нет регрессий в auth/admin
* документация/план актуализированы
