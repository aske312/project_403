# Project_403

Project_403 — приватный мессенджер, который постепенно развивается от MVP к полноценному сервису общения. Сейчас основной фокус проекта — удобный вход пользователей, понятная работа с профилем и подготовка пространства, где можно будет быстро начинать личные и групповые диалоги.

## Что уже работает

### Пользовательский доступ

- регистрация нового пользователя с базовой проверкой введенных данных;
- вход в аккаунт по email или пользовательскому тегу, чтобы не заставлять пользователя помнить только один вариант идентификатора;
- сохранение пользовательской сессии в браузере, чтобы после обновления страницы не начинать вход заново;
- автоматическая проверка текущего профиля при открытии приложения;
- понятное состояние истекшей сессии, когда пользователю нужно войти повторно;
- базовая защита от слишком частых попыток входа и регистрации.

### Основа мессенджера

- подготовлена основа пользовательского профиля, вокруг которого будут строиться диалоги и список контактов;
- подготовлена логика для будущих личных и групповых диалогов;
- подготовлена база для хранения истории сообщений, чтобы переписка могла сохраняться между сессиями;
- приложение уже развивается вокруг сценария мессенджера, а не просто отдельной формы входа.

## Что планируется дальше

### Пользователи

- список пользователей, с которыми можно начать общение;
- поиск по имени или пользовательскому тегу для быстрого нахождения нужного человека;
- выбор пользователя для начала личного чата без лишних промежуточных действий;
- более понятные состояния пустого списка, загрузки и ошибки.

### Чаты и сообщения

- создание личных чатов для общения один на один;
- создание групповых чатов для небольших обсуждений с несколькими участниками;
- отправка сообщений из выбранного диалога;
- сохранение истории, чтобы пользователь мог вернуться к прошлой переписке;
- подгрузка старых сообщений при открытии или прокрутке чата;
- понятные состояния отправки, доставки и ошибки сообщения.

### Общение в реальном времени

- обмен сообщениями без перезагрузки страницы;
- автоматическое восстановление соединения после временных проблем с сетью;
- индикатор online/offline, чтобы было понятно, доступен ли собеседник;
- индикатор набора текста для более живого диалога;
- статусы доставки и прочтения, чтобы пользователь видел, что происходит с отправленным сообщением.

### Удобство и надежность

- более аккуратная обработка ошибок, чтобы пользователь понимал, что пошло не так;
- стабильная работа с сессией между обновлениями страницы;
- улучшение интерфейса под ежедневное использование;
- подготовка приложения к более долгой и надежной работе;
- постепенное расширение сценариев общения без резкой смены привычного пользовательского опыта.
